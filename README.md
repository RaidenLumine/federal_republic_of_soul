# “斗罗联邦共和国” 档案

## 档案性质说明
- 本仓库仅仅是一个**个人云端存储库**（不能称之为一个 “项目”），用于存放虚构国家 “斗罗联邦共和国” 的静态设定档案，它源于**作者的各种奇思妙想**。
- 这仅作为一次世界构建的个人辅助备份与封存记录，无意构建活跃的社群或进行后续维护发布，**更遑论借此进行商业用途或侵权活动**。

### 灵感来源与融合框架（包括但不限于）
| 设定领域 | 主要借鉴 IP | 融合示例 |
| :--- | :--- | :--- |
| **主体框架** | 《斗罗大陆》系列 | 国名、魂导科技、史莱克学院等设定 |
| **元素体系** | 《原神》 | 提瓦特州、元素力应用、七国设定等设定 |
| **源石科技** | 《明日方舟》 | 明日州、源石技艺、移动城市等设定 |
| **魔法管理** | 《哈利·波特》等 | 魔法部（中央直属）、霍格沃茨魔法司等设定 |
| **太空科技** | 《崩坏：星穹铁道》等 | 云骑军建制、空间站与近地轨道设施命名 |
| **都市传说** | 《东方Project》系列 | 幻想乡、符卡规则管理 |

**典型融合案例**：
- **愚者公祠**：被设定为历史文化建筑，在特定情况下作为特别法庭使用。
- **魔法部**：作为帝国行政院下属的核心部门，下设多个 “司” 以垂直管理和规范不同文化背景下的魔法体系。

## 关于名词与翻译的说明

本设定集中所有名词（包括中英文）的选择，均服务于构建 “斗罗联邦共和国” 这一独特缝合世界的**整体氛围与作者的个人趣味**。

1.  **作者主权**：一个名词是沿用原作英文名、采用常见译名、或是进行重新定名，其**最终选择标准仅取决于作者本人的判断**，旨在使其更贴合本设定集的语境与风格，或者作者本人的风格。
2.  **设定优先**：所有名词的中英文形式，都应被视为本设定框架下的**正式官方名称**，它们共同构成了这个虚构国家的身份要素。
3.  **无标准答案**：这里不存在 “唯一正确” 的翻译，只存在 “在此设定中如此使用” 的名称。

**例**：“史莱克城”在本设定中对应的官方英文名为 **“Shrek City”**，“Endfield State”在本设定中对应的官方中文名为 **“未央州”**。这属于设定事实，而非翻译结论，无论原设定中该元素的英文或中文名称是什么。

### 辅助工具说明: `scripts/json_to_csv.py`
本仓库提供了一个简单的 Python 脚本，用于将 `data` 目录下的 `administrative_divisions.json` 快速转换为更便于在 Excel 等表格软件中阅读和二次编辑的 CSV 格式（带有可解决中文乱码的 BOM 头）。
- **运行环境**：需要安装 Python 3 环境（仅依赖标准库，无需安装第三方包）。
- **使用方法**：在此目录下打开终端，运行命令 `python scripts/json_to_csv.py`，即可在 `data` 目录中生成 `administrative_divisions.csv`。
