# 空间站设施

---

## 概述
- 斗罗联邦共和国在近地轨道和地球平流层部署了多个空间站和高空平台，形成了一个覆盖全球的空间基础设施网络。
- 这些设施承担着从军事防御、科学研究、深空探索到全球气象调控等多种关键功能，是国家安全、科技创新和国际合作的重要支柱。

---

## 主要空间站和高空平台

#### “安迪米昂” 空间站 (Endymion)
- 运行位置：地球静止轨道、学园都市上空
- 核心功能：
  - 云骑军太空军（联邦舰队）的**母港**，可停泊至少一个完整的舰队编制
  - 学园都市的**轨道校区**，开设零重力/特殊环境下的课程和研究项目
  - 国家太空十五的**战略指挥节点**之一
- 备注：
  - 总面积约 366.5 平方千米，其巨大体积在夜空中肉眼可见
  - 与学园都市的地面核心节点通过**太空电梯**连接，形成地面-轨道一体化的交通网络

#### “帝江” 空间站 (Dijiang)
- 运行位置：近地轨道
- 核心功能：
  - 深空探索前哨与在轨制造/维修中心
  - 国家行星际任务的主要出发与保障基地之一
- 备注：
  - 在**未央州帝江市**有一个地面核心节点，负责与空间站的对接和资源调配
  - 与 “集成工业系统” 在地面的核心节点联动，提供制造和维修支持

#### “黑塔” 空间站 (Herta)
- 运行位置：近地轨道
- 核心功能：
  - 太空科学研究平台、**“模拟宇宙”** 项目的核心设施
  - 国家行星际任务的主要出发与保障基地之一
- 备注：与圣芙蕾雅大学等研究机构紧密合作，承担大量科研项目

#### “逐界苍星” 空间站 (Terminus Astra)
- 运行位置：近地轨道
- 核心功能：
  - 星际探索的支持中心
  - 搭载同名超级计算机，提供强大算力支持星际航行计算、外世界环境模拟、深空导航等任务
  - 国家行星际任务的主要出发与保障基地之一
- 备注：通过 **“赫尔卡天梯”** 与**花仙州赫尔卡城**连接

#### “阿莱达” 高空平台 (Aleda)
- 运行位置：地球平流层
- 核心功能：
  - **全球气象干预**
  - **战略监视与支援平台**
  - 对特定区域的气候进行精确调控
  - 通信中继与情报收集
